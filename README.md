# 💼 Meu Portfólio — Elias Junior Cunha

Portfólio profissional desenvolvido com **HTML5**, **CSS3** e **JavaScript** puro, apresentando meus projetos e habilidades como Desenvolvedor Full Stack.

🔗 **[Ver online](https://eliajunior.github.io/meu-portfolio/)**

---

## 📋 Sobre o Projeto

Este portfólio foi desenvolvido como MVP (Produto Mínimo Viável) utilizando apenas tecnologias web fundamentais, sem frameworks ou bibliotecas externas.

### Funcionalidades

- ✅ Estrutura semântica HTML5 (`header`, `main`, `section`, `footer`)
- ✅ Layout responsivo com **Flexbox** e **CSS Grid**
- ✅ Responsividade para dispositivos móveis (media queries)
- ✅ Projetos renderizados **dinamicamente via JavaScript** a partir de um array de objetos
- ✅ Filtro de projetos por categoria (Web / Desktop)
- ✅ Menu mobile com toggle (hamburger)
- ✅ Scroll suave entre seções

---

## 🛠 Tecnologias Utilizadas

- **HTML5** — Estrutura e semântica
- **CSS3** — Estilização (Flexbox, Grid, Media Queries)
- **JavaScript** — Interatividade e renderização dinâmica

---

## 📁 Estrutura

```
├── index.html      → Página principal
├── style.css       → Estilização e responsividade
├── script.js       → Array de projetos e renderização no DOM
└── imgs/           → Imagens dos projetos
```
